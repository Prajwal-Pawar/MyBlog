# MyBlog Client

Welcome to MyBlog Client, a web application built with **React** and **Tailwind CSS**.

## Features

- MyBlog is a blogging site, where you can post articles and read others articles.
- You can comment on others articles to give them feedback.
- You can delete, edit your articles.
- You can edit/update your blog profile.

## Technologies used

- React
- Tailwindcss

## Libraries used

- react-router-dom : for routing
- axios : for making http requests from browsers to server
- dayjs : for formatting dates
- jwt-decode : for decoding JWT token we got from server
- react-hot-toast : for notification toasts
- react-icons : for icons

## Getting started

1. Clone this repo.
2. Navigate to project directory.
3. Install dependencies using `npm install`.
4. Start the development server with `npm start`.
5. Open your browser and visit <http://localhost:3000> to view the application.

> To run this application and use its feature properly, you need to also run **MyBlog Server**.

Feel free to explore the codebase and contribute to make MyBlog even better!
